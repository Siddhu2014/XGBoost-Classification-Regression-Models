# Tabular ML Foundations: Bank & House Projects

This repository contains my early work with **XGBoost**, focusing on solving tabular data challenges. These projects were developed as part of my journey toward the **International Olympiad in Artificial Intelligence (IOAI)**.

## 📊 Project Overview

### 🏦 Bank Marketing (Classification)
* **Problem:** Predicting if a client will subscribe to a term deposit.
* **Approach:** Used Gradient Boosting (XGBoost) to handle class imbalance and categorical features.
* **Outcome:** A high-precision classifier for marketing lead optimization.

### 🏠 House Prices (Regression)
* **Problem:** Estimating home values based on structural and neighborhood features.
* **Approach:** Implemented an XGBoost Regressor to capture non-linear relationships between house attributes.
* **Outcome:** A robust predictive model for real estate valuation.

## 🛠️ Tech Stack
* **Model:** XGBoost (Extreme Gradient Boosting)
* **Data Processing:** Pandas, NumPy
* **Environment:** VS Code, Jupyter Notebooks
